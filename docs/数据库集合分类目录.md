支持在数据中设置集合分类目录，实现对数据库中的集合分级分类管理。

collection_dir

| 字段名      | 标题名                                |
| ----------- | ------------------------------------- |
| name        | 系统名（英文）                        |
| title       | 显示名（中文）                        |
| full_name   | 完整名称                              |
| description | 备注                                  |
| level       | 分类的层级，从 1 开始。为了便于排序。 |
| order       | 在父节点下的排序                      |
| db.sysname  | 所属数据库系统名称                    |
| db.name     | 所属数据库名称                        |
| bucket      | 所属空间名称                          |
