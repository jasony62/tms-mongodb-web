# 接口清单

| 接口             | 方法 | 说明         |
| ---------------- | ---- | ------------ |
| /admin/db/list   | GET  | 所有的数据库 |
| /admin/db/create | POST | 新建数据库   |
| /admin/db/remove | GET  | 删除数据库   |
| /admin/db/top    | GET  | 数据库置顶   |

# 接口说明

## 新建数据库

**url**

> /admin/db/create

**参数**

| 名称 | 类型   | 说明                                                                         |
| ---- | ------ | ---------------------------------------------------------------------------- |
| db   | string | 数据库名称。库名必须以英文字母开头，可用英文字母或\_或数字组合，最长 64 位。 |
